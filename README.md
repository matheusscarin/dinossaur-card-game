<h1 align="center"> Jogo de Cartas de Dinossauro :sauropod: <h1>

Trata-se de um jogo de cartas semelhante ao famoso Super Trunfo, desenvolvido em Python como primeiro projeto pessoal durante a minha transição de carreira para Engenharia de Software.

## Objetivo desse Repositório :dart:

O objetivo desse projeto foi colocar em prática o aprendizado adquirido na Trilha de Conhcecimento: Linguagem de Programação Python da Fundação Bradesco - Escola Virtual. 

## Como Jogar :video_game:

1. Execute o jogo pelo terminal
2. O sorteio define quem começa (você ou a máquina)
3. A cada rodada, você vê sua carta e escolhe um atributo
4. O maior valor vence a rodada e fica com duas cartas
5. A carta **Super Trunfo** vence automaticamente qualquer outra
6. Ganha quem colecionar todas as cartas do adversário
